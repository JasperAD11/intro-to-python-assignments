# Intro to Python Assignments

This repository contains three introductory Python assignments completed as part of coursework in programming fundamentals.

## Overview

The assignments cover core Python concepts including:

- functions and basic program structure
- fundamental programming logic and problem solving
- object-oriented programming
- introductory data wrangling

Some of the later exercises are relatively advanced for an early-stage Python course and involve more structured problem solving and working with external data files.

## Repository contents

- `assignment_01_functions.ipynb`  
  First notebook assignment covering introductory Python concepts.

- `assignment_02_core_concepts.ipynb`  
  Second notebook assignment with more advanced programming exercises.

- `assignment_03_data_wrangling.ipynb`  
  Third notebook assignment, including data wrangling tasks.

- `wrangling_part_1.csv`  
  Data file used in Assignment 3.

- `wrangling_part_2.xlsx`  
  Data file used in Assignment 3.

- `wrangling_part_3.csv`  
  Data file used in Assignment 3.

- `wrangling_part_4.csv`  
  Data file used in Assignment 3.
